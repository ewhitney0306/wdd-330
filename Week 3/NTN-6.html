.querySelector() -use CSS notation find the first element in the document that matches that matches a CSS selector provided as an argument
.querySelectorAll() - uses CSS notation returns a node list of all the elements in the document that match the CSS query selector
The getAttribute() method returns the value of the attribute provided as an argument
The setAttribute can change the value of an element’s attributes
className property that allows the class of an element to be set directly
classList property is a list of all the classes an element has
add method can be used to add a class to an element without overwriting any classes that already exist
remove method will remove a specific class from an element
The toggle method is a particularly useful method that will add a class if an element doesn’t have it already, and remove the class if it does have it. It returns true if the class was added and false if it was removed.
contains method will check to see if an element has a particular class
createElement() method that takes a tag name as a parameter and returns that element
A text node can be created using the document.createTextNode() method. It takes a parameter, which is a string containing the text that goes in the node

function createElement (tag,text) {
const el = document.createElement(tag);
el.textContent = text;
return el
}
const aquaman = createElement('li','Aquaman');

insertBefore() method will place a new element before another element in the markup
An element can be removed from a page using the removeChild() method.
replaceChild() method can be used to replace one node with another.
The power of the innerHTML property becomes even more apparent if you want to insert a large amount of HTML into the document
getComputedStyle() that will retrieve all the style information of an element that is given as a parameter
