Javascript Modules

Background

dependent on import and export statements

Basic structure

  • index.html
  • main.js
  • modules
    • canvas.js
    • square.js

Export

  • functions
  • var
  • let
  • const
  • classes

Can't export inside a function

Use single export statement at the end

IE. export{name, draw, reportArea, reportPerimeter}

Import

import {name, draw, reportArea, reportPerimeter} from './modules/square.js';

Applying to html

script type='module' src='main.js'

NOTE

you can only use import and export inside modules

Module Rules

  • User strict mode automatically
  • are deferred automatically
  • only execute once
